# Undercover

Un jeu de rôle en ligne de style "Undercover" entièrement réinventé en HTML.

## Contenu
- `undercover.html` : page du jeu jouable dans le navigateur.

## Installation
Aucune installation nécessaire.

## Utilisation
1. Ouvrez `undercover.html` dans votre navigateur.
2. Suivez les instructions affichées pour créer des rôles, choisir des mots et démarrer la partie.

## À propos
Ce projet propose une interface immersive avec un dossier confidentiel et des mots secrets pour un jeu d'ambiance entre amis.
